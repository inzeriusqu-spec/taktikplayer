## 🚀 О проекте

**TaktikPlayer** - это современный лендинг для геймеров, предлагающий профессиональные тактики и стратегии для популярных киберспортивных игр. Проект создан для игроков в возрасте 16-35 лет, которые хотят улучшить свои навыки и достигать целей в 2 раза быстрее.

### Уникальное торговое предложение (УТП)
⚡ **Достигай целей в 2 раза быстрее** с профессиональными стратегиями от топ-игроков!

## ✨ Особенности

- 🎯 **Hero-секция** с анимированным фоном и призывом к действию
- 💎 **6 карточек преимуществ** с hover-анимациями и эффектами
- ⭐ **4 отзыва** реальных игроков с рейтингами
- ❓ **FAQ секция** с аккордеоном для частых вопросов
- 📝 **Форма подписки** с валидацией данных
- 🎨 **Геймерский дизайн** с неоновыми акцентами
- 📱 **Полная адаптивность** для всех устройств
- ✨ **Анимации при скролле** (fade-in эффекты)
- 🔄 **Плавная прокрутка** по якорным ссылкам
- 📊 **Анимированная статистика** в hero-секции
